🎮 Ball Game

A simple browser-based Ball Game built using HTML, CSS, and JavaScript. This project demonstrates basic game development concepts like object-oriented design, collision detection, and user interaction.

🚀 Features
🟢 Ball movement with collision detection
🎯 Paddle control using keyboard input
⚡ Smooth game loop and rendering
🎨 Styled UI using CSS
🧩 Modular code structure (separate JS files)

🛠️ Tech Stack
HTML – Structure of the game
CSS – Styling and layout
JavaScript – Game logic and interactivity


📂 Project Structure
Ball-Game/
│── index.html      # Main HTML file
│── styles.css      # Styling for game layout
│── script.js       # Main game logic
│── Ball.js         # Ball class (movement & collision)
│── Paddle.js       # Paddle class (controls & movement)
│── README.md       # Project documentation


🎮 How to Play
Open index.html in your browser
Use keyboard controls (e.g., arrow keys) to move the paddle
Prevent the ball from falling
Try to keep the ball in play as long as possible!


🧠 Concepts Used
Object-Oriented Programming (OOP)
Event Handling (keyboard input)
Collision Detection
Animation using JavaScript
